Techniquement [godot-fork] n'est pas réellement un fork comme l'entend GitHub, car j'aurais été contraint de laisser le dépôt public, ce qu'il ne faut pas pour empêcher un minimum la décompilation d'un jeu.

---

Recréation de zéro (sur la branche `main`)
```bash
git init godot-fork
cd godot-fork
git remote add upstream git@github.com:godotengine/godot
git remote add origin git@github.com:IdleClover/godot-fork.git
# Suivre les étapes pour créer un nouveau projet
```

[godot-fork]: https://github.com/IdleClover/godot-fork
